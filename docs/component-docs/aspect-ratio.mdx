# Aspect Ratio

The Aspect Ratio component is a molecule in the AliveUI component library. It helps maintain a consistent width-to-height ratio for content such as images, videos, and maps, preventing layout shifts as the page loads.

## Usage

```tsx
import { AspectRatio } from "@aliveui/ui";
import Image from "next/image";

export default function AspectRatioExample() {
  return (
    <AspectRatio ratio={16 / 9} className="bg-muted">
      <Image
        src="https://images.unsplash.com/photo-1588345921523-c2dcdb7f1dcd?w=800&dpr=2&q=80"
        alt="Photo by Drew Beamer"
        fill
        className="rounded-md object-cover"
      />
    </AspectRatio>
  );
}
```

## API Reference

The Aspect Ratio component is built on top of the Radix UI Aspect Ratio primitive.

### AspectRatio

| Property    | Type     | Default | Description                                    |
| ----------- | -------- | ------- | ---------------------------------------------- |
| `ratio`     | `number` | `1`     | The aspect ratio of the content                |
| `className` | `string` | -       | Additional CSS classes to add to the component |

## Examples

### Default 1:1 Ratio

```tsx
<AspectRatio className="bg-muted w-[400px]">
  <div className="flex items-center justify-center">
    <span className="text-muted-foreground">1:1 Aspect Ratio</span>
  </div>
</AspectRatio>
```

### 16:9 Ratio with Image

```tsx
<AspectRatio ratio={16 / 9} className="bg-muted w-[400px]">
  <Image
    src="https://images.unsplash.com/photo-1588345921523-c2dcdb7f1dcd?w=800&dpr=2&q=80"
    alt="Photo by Drew Beamer"
    fill
    className="rounded-md object-cover"
  />

```

### 4:3 Ratio with Video

```tsx
<AspectRatio ratio={4 / 3} className="w-[400px]">
  <video
    controls
    className="h-full w-full rounded-md object-cover"
    src="https://examples.com/video.mp4"
  />
</AspectRatio>
```

## Accessibility

The Aspect Ratio component is primarily a layout utility and doesn't have specific accessibility concerns. However, it's important to ensure that:

- Images within the Aspect Ratio component have proper `alt` text
- Videos have captions and proper controls
- Content within the component remains accessible to assistive technologies

## Implementation Details

The Aspect Ratio component is built on top of Radix UI's Aspect Ratio primitive, ensuring consistent behavior across browsers. It maintains the specified aspect ratio by using CSS to reserve the appropriate space, preventing layout shifts as content loads. This is particularly useful for responsive designs where maintaining proportions is important.
