# Menubar

The Menubar component is a molecule in the AliveUI component library. It provides a horizontal menu system commonly used in application headers, offering dropdown menus with various item types and keyboard navigation.

## Usage

```tsx
import {
  Menubar,
  MenubarMenu,
  MenubarTrigger,
  MenubarContent,
  MenubarItem,
  MenubarSeparator,
  MenubarShortcut,
} from "@aliveui/ui";

export function MenubarDemo() {
  return (
    <Menubar>
      <MenubarMenu>
        <MenubarTrigger>File</MenubarTrigger>
        <MenubarContent>
          <MenubarItem>
            New Tab <MenubarShortcut>⌘T</MenubarShortcut>
          </MenubarItem>
          <MenubarItem>
            New Window <MenubarShortcut>⌘N</MenubarShortcut>
          </MenubarItem>
          <MenubarSeparator />
          <MenubarItem>
            Print <MenubarShortcut>⌘P</MenubarShortcut>
          </MenubarItem>
        </MenubarContent>
      </MenubarMenu>
      <MenubarMenu>
        <MenubarTrigger>Edit</MenubarTrigger>
        <MenubarContent>
          <MenubarItem>
            Undo <MenubarShortcut>⌘Z</MenubarShortcut>
          </MenubarItem>
          <MenubarItem>
            Redo <MenubarShortcut>⇧⌘Z</MenubarShortcut>
          </MenubarItem>
          <MenubarSeparator />
          <MenubarItem>
            Cut <MenubarShortcut>⌘X</MenubarShortcut>
          </MenubarItem>
          <MenubarItem>
            Copy <MenubarShortcut>⌘C</MenubarShortcut>
          </MenubarItem>
          <MenubarItem>
            Paste <MenubarShortcut>⌘V</MenubarShortcut>
          </MenubarItem>
        </MenubarContent>
      </MenubarMenu>
      <MenubarMenu>
        <MenubarTrigger>View</MenubarTrigger>
        <MenubarContent>
          <MenubarItem>
            Zoom In <MenubarShortcut>⌘+</MenubarShortcut>
          </MenubarItem>
          <MenubarItem>
            Zoom Out <MenubarShortcut>⌘-</MenubarShortcut>
          </MenubarItem>
          <MenubarSeparator />
          <MenubarItem>
            Toggle Fullscreen <MenubarShortcut>F11</MenubarShortcut>
          </MenubarItem>
        </MenubarContent>
      </MenubarMenu>
    </Menubar>
  );
}
```

## API Reference

The Menubar component is built on top of Radix UI's Menubar primitive and consists of several parts.

### Menubar

The root component that contains the menubar items.

| Property    | Type     | Default | Description            |
| ----------- | -------- | ------- | ---------------------- |
| `className` | `string` | -       | Additional CSS classes |

### MenubarMenu

A container for a single menu and its content.

### MenubarTrigger

The button that opens a menu.

| Property    | Type     | Default | Description            |
| ----------- | -------- | ------- | ---------------------- |
| `className` | `string` | -       | Additional CSS classes |

### MenubarContent

The component that contains menu items.

| Property      | Type     | Default   | Description                                   |
| ------------- | -------- | --------- | --------------------------------------------- |
| `align`       | `string` | `"start"` | Alignment of the menu relative to the trigger |
| `alignOffset` | `number` | `-4`      | Offset in pixels from the aligned edge        |
| `sideOffset`  | `number` | `8`       | Distance in pixels from the trigger           |
| `className`   | `string` | -         | Additional CSS classes                        |

### MenubarItem

A clickable menu item.

| Property    | Type      | Default | Description                                      |
| ----------- | --------- | ------- | ------------------------------------------------ |
| `inset`     | `boolean` | `false` | Whether the item should have padding on the left |
| `disabled`  | `boolean` | `false` | Whether the item is disabled                     |
| `className` | `string`  | -       | Additional CSS classes                           |

### MenubarCheckboxItem

A menu item with a checkbox.

| Property          | Type       | Default | Description                                     |
| ----------------- | ---------- | ------- | ----------------------------------------------- |
| `checked`         | `boolean`  | -       | The controlled checked state of the checkbox    |
| `onCheckedChange` | `function` | -       | Event handler called when checked state changes |
| `className`       | `string`   | -       | Additional CSS classes                          |

### MenubarRadioGroup

A group of radio items where only one can be selected.

| Property        | Type       | Default | Description                                     |
| --------------- | ---------- | ------- | ----------------------------------------------- |
| `value`         | `string`   | -       | The controlled value of the selected radio item |
| `onValueChange` | `function` | -       | Event handler called when value changes         |

### MenubarRadioItem

A menu item that can be selected from a group of radio items.

| Property    | Type     | Default | Description                        |
| ----------- | -------- | ------- | ---------------------------------- |
| `value`     | `string` | -       | The unique value of the radio item |
| `className` | `string` | -       | Additional CSS classes             |

### MenubarSubTrigger

The trigger for a submenu.

| Property    | Type      | Default | Description                                         |
| ----------- | --------- | ------- | --------------------------------------------------- |
| `inset`     | `boolean` | `false` | Whether the trigger should have padding on the left |
| `className` | `string`  | -       | Additional CSS classes                              |

### MenubarSubContent

The content for a submenu.

| Property    | Type     | Default | Description            |
| ----------- | -------- | ------- | ---------------------- |
| `className` | `string` | -       | Additional CSS classes |

### MenubarSeparator

A visual separator between menu items.

| Property    | Type     | Default | Description            |
| ----------- | -------- | ------- | ---------------------- |
| `className` | `string` | -       | Additional CSS classes |

### MenubarLabel

A non-interactive label for a group of items.

| Property    | Type      | Default | Description                                       |
| ----------- | --------- | ------- | ------------------------------------------------- |
| `inset`     | `boolean` | `false` | Whether the label should have padding on the left |
| `className` | `string`  | -       | Additional CSS classes                            |

### MenubarShortcut

Displays keyboard shortcut hints.

| Property    | Type     | Default | Description            |
| ----------- | -------- | ------- | ---------------------- |
| `className` | `string` | -       | Additional CSS classes |

### With Checkboxes

```tsx

function CheckboxMenubarExample() {
  const [showStatusBar, setShowStatusBar] = useState(true);
  const [showActivityBar, setShowActivityBar] = useState(false);
  const [showPanel, setShowPanel] = useState(false);

  return (
    <Menubar>
      <MenubarMenu>
        <MenubarTrigger>View</MenubarTrigger>
        <MenubarContent>
          <MenubarCheckboxItem
            checked={showStatusBar}
            onCheckedChange={setShowStatusBar}
          >
            Status Bar
          </MenubarCheckboxItem>
          <MenubarCheckboxItem
            checked={showActivityBar}
            onCheckedChange={setShowActivityBar}
          >
            Activity Bar
          </MenubarCheckboxItem>
          <MenubarCheckboxItem
            checked={showPanel}
            onCheckedChange={setShowPanel}
          >
            Panel
          </MenubarCheckboxItem>
        </MenubarContent>
      </MenubarMenu>
    </Menubar>
  );
}
```

## Accessibility

The Menubar component follows these accessibility best practices:

- Keyboard navigable: can be operated with arrows, tab, space, enter, and escape keys
- Focus management: focus is trapped inside open menus
- Supports typeahead for quick keyboard navigation
- Proper ARIA roles, states, and properties
- Clear visual indication of focus and selected states
- Follows WAI-ARIA design patterns for menus
- Submenus properly indicate their parent relationships
- Keyboard shortcuts are visible and accessible
- Disabled states appropriately prevent interaction

## Implementation Details

The Menubar component is built on top of Radix UI's Menubar primitive, providing a robust and accessible menubar implementation. It is styled with Tailwind CSS and includes features like:

- Multiple menu types (standard, checkbox, radio)
- Submenu support for hierarchical navigation
- Keyboard shortcuts display
- Disabled state styling
- Focus and hover states
- Proper animation for opening and closing menus

Menubars are ideal for applications with multiple features that need to be organized in a hierarchical structure. They're commonly used in desktop-style applications, content editors, and administrative interfaces.
