# Navigation Menu

The Navigation Menu component is a molecule in the AliveUI component library. It provides a responsive navigation menu system with dropdown menus, flyout menus, and keyboard navigation support.

## Usage

```tsx
import {
  NavigationMenu,
  NavigationMenuContent,
  NavigationMenuItem,
  NavigationMenuLink,
  NavigationMenuList,
  NavigationMenuTrigger,
  navigationMenuTriggerStyle,
} from "@aliveui/ui";

export function NavigationMenuDemo() {
  return (
    <NavigationMenu>
      <NavigationMenuList>
        <NavigationMenuItem>
          <NavigationMenuTrigger>Getting started</NavigationMenuTrigger>
          <NavigationMenuContent>
            {/* Content for the dropdown */}
          </NavigationMenuContent>
        </NavigationMenuItem>

        <NavigationMenuItem>
          <NavigationMenuLink
            className={navigationMenuTriggerStyle()}
            href="/docs"
          >
            Documentation
          </NavigationMenuLink>
        </NavigationMenuItem>
      </NavigationMenuList>
    </NavigationMenu>
  );
}
```

## API Reference

### NavigationMenu

The root component of the navigation menu.

```tsx
<NavigationMenu>{/* NavigationMenuList */}</NavigationMenu>
```

### NavigationMenuList

Contains the list of navigation items.

```tsx
<NavigationMenuList>{/* NavigationMenuItems */}</NavigationMenuList>
```

### NavigationMenuItem

An item in the navigation menu.

```tsx
<NavigationMenuItem>
  {/* NavigationMenuTrigger or NavigationMenuLink */}
</NavigationMenuItem>
```

### NavigationMenuTrigger

A trigger button that opens a dropdown menu.

```tsx
<NavigationMenuTrigger>Label</NavigationMenuTrigger>
```

### NavigationMenuContent

The content of a dropdown menu.

```tsx
<NavigationMenuContent>{/* Dropdown content */}</NavigationMenuContent>
```

### NavigationMenuLink

A link in the navigation menu.

```tsx
<NavigationMenuLink href="/path">Label</NavigationMenuLink>
```

### navigationMenuTriggerStyle

A style function for links that should look like triggers.

```tsx
<NavigationMenuLink className={navigationMenuTriggerStyle()}>
  Label
</NavigationMenuLink>
```

### NavigationMenuIndicator

An indicator that shows which item is active.

```tsx
<NavigationMenuIndicator />
```

### NavigationMenuViewport

The viewport for displaying dropdown content.

```tsx
<NavigationMenuViewport />
```

## Accessibility

The Navigation Menu component is built on top of Radix UI's NavigationMenu primitive, ensuring excellent accessibility:

- Follows the [WAI-ARIA Navigation Menu Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/menubar/)
- Full keyboard navigation support
- Proper ARIA attributes for screen readers
- Focus management for dropdown content
- Properly labeled menu items

## Implementation Details

The Navigation Menu component is built on Radix UI's NavigationMenu primitive, providing a solid foundation for building accessible navigation systems.

The component supports:

- Dropdowns with rich content
- Keyboard navigation
- Responsive layouts
- Custom styling
- Animated transitions

The styling is handled with Tailwind CSS, making it easy to customize to match your design system. The `navigationMenuTriggerStyle` function provides consistent styling for links that should look like triggers.
