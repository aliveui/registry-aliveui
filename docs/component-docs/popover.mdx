# Popover

The Popover component is a molecule in the AliveUI component library. It provides a floating content panel that appears when a trigger element is clicked, displaying additional information or controls.

## Usage

```tsx
import { Popover, PopoverContent, PopoverTrigger } from "@aliveui/ui";

export function PopoverDemo() {
  return (
    <Popover>
      <PopoverTrigger>Open Popover</PopoverTrigger>
      <PopoverContent>
        <p>This is the content of the popover.</p>
      </PopoverContent>
    </Popover>
  );
}
```

## API Reference

### Popover

The root component for the popover.

```tsx
<Popover>{/* PopoverTrigger and PopoverContent */}</Popover>
```

### PopoverTrigger

The element that triggers the popover when clicked.

```tsx
<PopoverTrigger>Open Popover</PopoverTrigger>
```

### PopoverContent

The content that appears when the popover is open.

```tsx
<PopoverContent>{/* Popover content */}</PopoverContent>
```

**Props**

- `align`: The alignment of the popover relative to the trigger. Options: `"start"`, `"center"`, `"end"`. Default: `"center"`.
- `sideOffset`: The distance between the trigger and the popover. Default: `4`.
- All other props are passed to the underlying radix-ui component.

```tsx

          Set the dimensions for the layer.
        </p>
      </div>
      <div className="grid gap-2">
        <div className="grid grid-cols-3 items-center gap-4">
          <Label htmlFor="width">Width</Label>

        <div className="grid grid-cols-3 items-center gap-4">

          <Input id="height" defaultValue="25px" className="col-span-2 h-8" />
```


### Popover with Custom Positioning

```tsx
<Popover>
  <PopoverTrigger asChild>
    <Button variant="outline">Show at Bottom Start</Button>
  </PopoverTrigger>
  <PopoverContent align="start" sideOffset={8}>
    <p>This popover is aligned to the start and has a side offset of 8px.</p>
  </PopoverContent>
</Popover>
```

### Popover with Close Button


## Accessibility

The Popover component is built on top of Radix UI's Popover primitive, ensuring excellent accessibility features:

- Follows the [WAI-ARIA Design Pattern for Popover](https://www.w3.org/WAI/ARIA/apg/patterns/dialogmodal/)
- Manages focus correctly, trapping it within the popover when open
- Proper ARIA attributes for screen readers
- Keyboard navigable
- Closes when pressing Escape
- Implements proper click-outside behavior

## Implementation Details

The Popover component is built with Radix UI's Popover primitive and Tailwind CSS. It provides a consistent way to display additional content that doesn't fit within the main flow of a page.

The component supports:

- Custom positioning
- Animations
- Various styling options
- Focus management
- Keyboard interactions

The `PopoverContent` component uses Portal to render outside the DOM hierarchy, ensuring proper stacking context.
