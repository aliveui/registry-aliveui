# Skeleton

The Skeleton component is an atom in the AliveUI component library. It's used to create placeholder loading states that visually represent content that's still loading.

## Usage

```tsx
import { Skeleton } from "@aliveui/ui";

export default function SkeletonExample() {
  return (
    <div className="flex items-center space-x-4">
      <Skeleton className="h-12 w-12 rounded-full" />
      <div className="space-y-2">
        <Skeleton className="h-4 w-[250px]" />
        <Skeleton className="h-4 w-[200px]" />
      </div>
    </div>
  );
}
```

## API Reference

### Properties

The Skeleton component extends the HTML div element and accepts all div properties.

| Property    | Type     | Default | Description                                     |
| ----------- | -------- | ------- | ----------------------------------------------- |
| `className` | `string` | -       | Additional CSS classes to apply to the skeleton |

### Conditional Skeleton

```tsx
import { useState, useEffect } from "react";

function ConditionalSkeleton() {
  const [isLoading, setIsLoading] = useState(true);
  const [data, setData] = useState(null);

  useEffect(() => {
    // Simulate data fetching
    const timer = setTimeout(() => {
      setData("Content loaded!");
      setIsLoading(false);
    }, 3000);

    return () => clearTimeout(timer);
  }, []);

  return (
    <div className="h-[100px] w-[300px] rounded-md border p-4">
      {isLoading ? (
        <div className="space-y-2">
          <Skeleton className="h-4 w-full" />
          <Skeleton className="h-4 w-4/5" />
          <Skeleton className="h-4 w-3/5" />
        </div>
      ) : (
        <div>{data}</div>
      )}
    </div>
  );
}
```

## Accessibility

The Skeleton component follows accessibility best practices:

- Uses appropriate ARIA attributes to communicate loading state
- The pulse animation is subtle to avoid causing issues for users with vestibular disorders or photosensitivity
- The animation can be disabled using the prefers-reduced-motion media query

## Implementation Details

The Skeleton component is a simple div styled with Tailwind CSS to have a pulsing animation. It's a lightweight component that provides a uniform loading state throughout the application. The component uses the `animate-pulse` class from Tailwind CSS for the animation.
