# Tabs

The Tabs component is a molecule in the AliveUI component library. It provides a tabbed interface for switching between different sections of content, allowing users to navigate through multiple panes within a compact space.

## Usage

```tsx
import { Tabs, TabsList, TabsTrigger, TabsContent } from "@aliveui/ui";

export function TabsDemo() {
  return (
    <Tabs defaultValue="account" className="w-[400px]">
      <TabsList>
        <TabsTrigger value="account">Account</TabsTrigger>
        <TabsTrigger value="password">Password</TabsTrigger>
      </TabsList>
      <TabsContent value="account">
        <p>Account settings tab content</p>
      </TabsContent>
      <TabsContent value="password">
        <p>Password settings tab content</p>
      </TabsContent>
    </Tabs>
  );
}
```

## API Reference

### Tabs

The root component that manages the tab state.

```tsx
<Tabs defaultValue="account">{/* TabsList and TabsContent */}</Tabs>
```

**Props**

- `defaultValue`: The initial value of the selected tab.
- `value`: The controlled value of the selected tab.
- `onValueChange`: A callback function called when the selected tab changes.

### TabsList

The container for the tab triggers.

```tsx
<TabsList>{/* TabsTrigger components */}</TabsList>
```

### TabsTrigger

The button that activates a specific tab.

```tsx
<TabsTrigger value="account">Account</TabsTrigger>
```

**Props**

- `value`: A unique identifier for the tab. This should match the `value` of the corresponding `TabsContent`.

### TabsContent

The content that is displayed when the corresponding tab is active.

```tsx
<TabsContent value="account">
  <p>Account settings tab content</p>
</TabsContent>
```

**Props**

- `value`: A unique identifier for the tab content. This should match the `value` of the corresponding `TabsTrigger`.

## Accessibility

The Tabs component is built on top of [Radix UI's Tabs primitive](https://www.radix-ui.com/primitives/docs/components/tabs), which ensures excellent accessibility:

- Follows the [WAI-ARIA Tabs Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/tabpanel/)
- Keyboard navigation with arrow keys to switch between tabs
- Tab and Shift+Tab for navigating through the tabbed interface
- Screen reader announcements when tabs change
- Proper ARIA roles, states, and properties

## Implementation Details

The Tabs component is built using Radix UI's Tabs primitive and Tailwind CSS for styling. It provides a simple yet powerful way to organize content into tabbed sections, improving the user experience when dealing with multiple sections of related content.

The component supports:

- Controlled and uncontrolled modes
- Keyboard navigation
- Customizable styling
- Screen reader accessibility
- Responsive layouts

Tabs are useful for:

- Settings pages with multiple categories
- Product details with different sections
- Dashboard widgets with multiple data views
- Form wizards and multi-step processes
